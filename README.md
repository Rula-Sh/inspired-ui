# Inspired UI 🌟

collection of interactive UI components and mini-projects inspired by modern web design trends. Each component is built with pure HTML, CSS, and JavaScript, making them easy to understand, modify, and integrate into your own projects.

## Features ✨

- **Central Navigation Page** - Main projects page that navigates to all mini-projects
- **Isolated Code** - Each mini-project is self-contained with its own HTML, CSS, and JavaScript
- **No Dependencies** - No frameworks or libraries, just vanilla code
- **Responsive Designs** - Components adapt to different screen sizes
- **Modern UI Trends** - Collection of modern UI trends and interactions
- **Easy Integration** - Grab any component and drop it into your existing projects
